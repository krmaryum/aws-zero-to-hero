# README – Day 1 AWS Hands-on Practice MCQs

## Overview

This HTML quiz contains 25 multiple-choice questions based on Day 1 AWS hands-on practice notes.

Topics covered:

- AWS CLI configure
- EC2 instance creation
- IAM user, access key, and S3 mini project
- S3 CLI bucket access, prefix listing, recursive listing, and file upload

## Features

- 25 MCQs
- 25-minute timer
- Score checking
- Answer checking
- Short explanation for each question
- Topic-wise result breakdown
- Re-attempt button
- Answer choices shuffle on each re-attempt

## How to Use

1. Download the HTML file.
2. Open it in any web browser.
3. Select one answer for each question.
4. Click **Submit Quiz**.
5. Review explanations and topic-wise score.
6. Click **Re-attempt Quiz** to try again with shuffled answer choices.

## Suggested Goal

Try to score:

```text
80% or above
```

If the score is below 80%, review the related study notes and re-attempt.
