# Day 1 AWS MCQs – README

## Overview

This HTML quiz file contains 25 multiple-choice questions for Day 1 of the 7 Days of AWS Challenge.

It covers:

- AWS pricing models
- On-premises, Cloud, and Hybrid models
- IaaS, PaaS, and SaaS service models
- AWS history and key milestones

## How to Use

1. Download the HTML file.
2. Open it in any web browser.
3. Answer all 25 questions.
4. Click **Submit Quiz**.
5. Review your score and explanations.
6. Retake the quiz if needed.

## Quiz Features

- 25 MCQs
- 25-minute timer
- Answer checking
- Score calculation
- Short explanation for each question
- Show answers button
- Retake quiz button

Try to score at least:

80% or above
